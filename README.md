Livewire practice with livewire table and CRUD functionalities.

Integrated with:
> WireUI
> Nprogress
> Laravel Breeze

===================================================

Installation:

composer install

cp .env.example .env

php artisan key:generate

npm install

npm run dev

php artisan migrate

php artisan db:seed

php artisan serve
